**Bidhan Multi-Tool Pro** is a high-precision, professional-grade security and utility toolkit designed for modern web security management. It combines advanced cryptographic logic with a premium **Gemini-inspired Glassmorphism UI** to provide an unmatched user experience.

## 🚀 Core Features

### 🔒 2FA Authenticator
- **Precision Sync:** High-accuracy TOTP generation synchronized with global clock cycles.
- **Visual Timer:** Real-time countdown bar with dynamic warning states.
- **Activity Log:** Persistent tracking of secret keys for rapid retrieval.

### 🔑 Pass Pro (Advanced Generator)
- **Cryptographic Entropy:** Generates high-entropy passwords with custom length and character sets.
- **History Tracking:** Automatic logging of the last 3 generated passwords with easy-copy functionality.
- **Secure Vault Feel:** Designed with a deep-blue/gold security aesthetic.

### 📧 Email Gen (Identity Protocol)
- **Localized Profiles:** Generate identities across Bangladesh, USA, India, Pakistan, Vietnam, and Indonesia.
- **Smart Tracking:** Used emails are automatically highlighted and tracked to prevent duplication.
- **Global Domains:** Support for Gmail, Yahoo, Outlook, Hotmail, and iCloud protocols.

## ✨ Premium Experience

- **Gemini UI:** Translucent, fluid elements with violet-to-blue neon glow effects.
- **Developer Profile Card:** Interactive hover-to-view developer info with direct GitHub integration.
- **Split-Segment Navigation:** Professional animated navigation bar for seamless tool switching.
- **Responsive & Dark Optimized:** Built for high-end performance on all screen sizes.

## 🛠️ Technical Stack

- **Frontend:** HTML5, Tailwind CSS (JIT Engine)
- **Logic:** Vanilla JavaScript (ES6+ Architecture)
- **Cryptographic Library:** [Otplib](https://github.com/yeojz/otplib) for RFC 6238 compliance.
- **UI Design:** Modern Glassmorphism with Bezier-curve animations.
